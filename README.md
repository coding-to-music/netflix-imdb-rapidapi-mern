# WatchList

![](gif/rkbqCHoObe.gif)

## About The Project

You can keep track of all films and TV shows that you have watched, watched or plan to watch.

Now you will not forget to watch the new episode of your favorite TV series
## Technologies
* **react : 17.0.2**
* **redux : 4.1.1**
* **react-redux : 7.2.4**
* **axios : 0.21.1**
* **node-sass: 6.0.1**
* **express": 4.17.1**
* **nedb": 1.8.0**
* **google-auth-library": 7.6.2**
* **Movie Database (IMDB Alternative) By RapidAPI**
* **Material UI**

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm test`

Launches the test runner in the interactive watch mode.\

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.


### `npm run lint `

Run lint checks.

### `npm run lint:fix`

Fix lint issues.

### `npm server:start`
Run backend local server.
